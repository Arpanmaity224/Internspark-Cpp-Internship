# Task 3 - Log Rotator

## Description
This program reads data from sample_log.txt and copies it to rotated_log.txt.

## Features
- Reads a log file
- Creates a rotated log file
- Handles file open errors
- Displays success message

## Compile
g++ log_rotator.cpp -o log_rotator

## Run
./log_rotator

## Sample Output
Reading sample_log.txt...
Writing to rotated_log.txt...
Log rotation completed successfully!
